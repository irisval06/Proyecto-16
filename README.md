# Proyecto-16
Análisis de Datos de una Plataforma de Libros (SQL)
Este proyecto consiste en el análisis de una base de datos relacional perteneciente a una plataforma digital de libros, desarrollada en el contexto del crecimiento del consumo de lectura en línea tras la pandemia de COVID-19. El objetivo principal es extraer información estratégica a partir de datos sobre libros, autores, editoriales, calificaciones y reseñas de usuarios.
La base de datos incluye múltiples tablas relacionadas que permiten explorar el comportamiento de los usuarios, el rendimiento de los libros y la actividad editorial dentro de la plataforma.
Objetivo
El propósito del análisis es generar insights que ayuden a construir una propuesta de valor para un nuevo producto digital enfocado en lectores, mediante el uso de consultas SQL para responder preguntas de negocio clave.
Actividades realizadas
Exploración inicial de la base de datos y comprensión del modelo relacional.
Análisis de tablas: libros, autores, editoriales, calificaciones y reseñas.
Desarrollo de consultas SQL para resolver problemas analíticos, incluyendo:
Número de libros publicados después del año 2000.
Cálculo de reseñas y calificación promedio por libro.
Identificación de la editorial con mayor número de libros extensos (más de 50 páginas).
Determinación del autor con mayor calificación promedio (considerando libros con al menos 50 reseñas).
Análisis del comportamiento de usuarios con alta actividad de calificación y escritura de reseñas.
Interpretación de resultados y extracción de conclusiones orientadas al negocio.
Herramientas utilizadas
SQL (consultas de agregación, filtrado y joins)
Base de datos relacional
Jupyter Notebook (para ejecución y visualización de resultados)
Pandas (solo para visualización de resultados)
Resultados esperados
El análisis permite identificar tendencias de consumo de lectura, autores y editoriales con mejor desempeño, así como patrones de comportamiento de usuarios activos, proporcionando información clave para la toma de decisiones en el desarrollo de nuevos productos digitales.
